# Все делители

> Time limit: 1000 ms \
> Memory Limit: 131072 kB

## Входные данные

Одно положительное число n (n ≤ 10^9).

## Выходные данные

Все делители n в порядке возрастания

## Пример

| Входные данные | :::      |
|:---------------|:---------|
| 10             | -------- |
| 36             |          |

| Выходные данные      | :::    |
|----------------------|:-------|
| 1 2 5 10             | ------ |
| 1 2 3 4 6 9 12 18 36 |        |
