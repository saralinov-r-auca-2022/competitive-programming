# Соревнование последовательностей

> Time limit: 1000 ms \
> Memory Limit: 131072 kB

Завтра Зия примет участие в соревновании последовательностей. Число x ≥ 0
называется вершиной некоторой последовательности, если
последовательность 1, 2, 3, ..., x - 1, x, x - 1, ..., 3, 2, 1
является подпоследовательностью этой последовательности. Силой каждой
последовательности считается ее наибольшая вершина.

Завтра все ученики пойдут на соревнования и победителем станет обладателем
самой сильной последовательности. Зия имеет последовательность
a[1], a[2], a[3], ..., a[n]. Он хочет захватить систему
подсчета очков на соревнованиях и удалить из нее эпизоды с
большей силой, чем он сам. Однако Зия не знает силы
собственной последовательности, но очень хочет победить.
Помогите ему рассчитать силу его собственной последовательности.

## Входные данные

Первая строка: количество n (1 ≤ n ≤ 10^5) чисел последовательности Зии
Вторая строка: n чисел a[i] (1 ≤ a[i] ≤ 10^5) - элементы последовательности.

## Выходные данные

Одно число - сила данной последовательности.

## Пример

| Входные данные     | Выходные данные |
|:-------------------|:----------------|
| 2 <br/> 2 10       | 0               |
| 3 <br/> 1 2 3      | 1               |
| 5 <br/> 1 10 2 3 1 | 2               |
